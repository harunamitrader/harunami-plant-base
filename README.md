# harunami-plant-base

harunami Plant Base の GitHub Pages 用静的サイトです。

## 公開内容

- 5大植物ホルモンの基礎
- オーキシン、ジベレリン、サイトカイニン、エチレン、アブシシン酸の代表作用
- 果樹・庭木の管理や剪定にどう生かすか
- 図解 4 枚を含む単一ページ構成
- 黒松の剪定ガイド
- 春の芽数調整、初夏のみどり摘み、秋冬のもみあげ・透かし剪定
- 黒松ページ用の図解 3 枚

## ページ

- `index.html`
  5大植物ホルモンの基礎と果樹・庭木管理
- `kuromatsu-pruning.html`
  黒松の剪定ガイド

## ローカル確認

PowerShell 例:

```powershell
py -m http.server 8765 --bind 127.0.0.1
```

その後、`http://127.0.0.1:8765/` を開く。
