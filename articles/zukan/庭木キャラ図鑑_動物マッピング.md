# 庭木キャラ図鑑 動物マッピング（ゆるキャラ路線）

- 目的: 各樹種を「動物（種類は自由）」のゆるキャラにする。猫限定をやめ、動物そのものが変わることで自然に差別化する。
- ルール: 1 動物につき最大 3 樹種まで。被りをできるだけ散らす。
- 選定軸: 葉・花・実・樹皮の色や形、樹形、季節感、香り、和洋の印象、名前の語呂、生息環境の連想。
- 対象: `庭木キャラ図鑑_樹種一覧.md` の 132 樹種。
- スタイル: 丸っこく親しみやすいゆるキャラ風デフォルメ。リアル寄りにしない。
- 旧 `庭木キャラ図鑑_猫品種マッピング.md`（TICA 猫品種版）はこの動物版に置き換え。猫版ファイルは削除せず参考として残す。
- 統合履歴: 旧 `タラノキ2`（ヤマアラシ）→ タラノキ（ハリネズミ）に統合 / 旧 `ニオイヒバ2`（デグー）→ ニオイヒバ（モルモット）に統合。134→132。
